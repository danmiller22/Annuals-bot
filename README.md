# Annual Inspection Telegram Bot

Минимальный бот для мониторинга annual inspection полуприцепов:

- слушает сообщения в Telegram-группе,
- ищет в тексте паттерн `PLATE YYYY-MM-DD` (номер + дата),
- по каждому полуприцепу хранит только самую свежую дату,
- раз в день проверяет сроки и за 30 дней до истечения шлёт предупреждение в группу.

## Локальный запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Указать токен бота одним из способов:
   - задать переменную окружения `TELEGRAM_TOKEN`, или
   - вписать токен вместо `YOUR_TELEGRAM_BOT_TOKEN_HERE` в `bot.py`.

3. Запустить бота:

```bash
python bot.py
```

В группе:
- добавить бота как участника,
- один раз отправить `/start`.
